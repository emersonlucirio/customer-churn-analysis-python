# Customer Churn Analysis with Python

## 📌 Project Overview
This project analyzes customer cancellation behavior using Python.  
The goal is to identify patterns and factors that influence customer churn.

## 🎯 Business Problem
High customer cancellation rates can negatively impact revenue.  
This analysis aims to identify the main variables associated with churn and simulate improvement scenarios.

## 🛠️ Technologies Used
- Python
- Pandas
- Plotly
- CSV data handling

## 🗄️ Database & Data Concepts Applied
- Data cleaning (removal of null values)
- Column filtering
- Exploratory Data Analysis (EDA)
- Percentage and categorical analysis
- Data visualization
- Scenario simulation with conditional filtering

## 📊 Key Insights
- Monthly contracts showed higher cancellation rates.
- Customers with more than 4 call center interactions had higher churn probability.
- Customers with payment delays over 20 days were more likely to cancel.

## 💡 Business Recommendations
- Encourage long-term contracts
- Improve customer support efficiency
- Implement proactive payment follow-up policies

## 🚀 Future Improvements
